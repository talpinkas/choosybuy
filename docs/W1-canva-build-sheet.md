## הסצנות (ציר זמן)

### סצנה 1 (~8ש) — חיפוש אינסופי + בזבוז זמן [אפור]
- רקע: אפור בהיר.
- **מוקאפ טלפון** במרכז (חפש באלמנטים "phone mockup").
- בתוך המסך: רשת אייקוני בגדים אפורים (חפש "t-shirt icon", "clothes icon"). קבץ אותם
  והחל אנימציית **Pan / Rise** כלפי מעלה כדי לדמות גלילה אינסופית; הגדל מהירות לקראת הסוף.
- **שעון מסתובב** בפינה (חפש סטיקר מונפש "spinning clock" / "clock gif").
- **חלון עם שמש→ירח** (חפש "day to night" / "sun moon" סטיקר) — להמחשת זמן שעובר.
- אנימציית כניסה לכל: Fade/Rise.

### סצנה 2 (~6ש) — שיא העומס [אפור]
- ערימת אייקוני בגדים גדלה סביב הטלפון.
- דמות/פרצוף עייף (חפש "tired", "stressed" illustration, בגוון אפור).
- החל על הטלפון אנימציית הדגשה **Shake / Wiggle**. תחושת מצוקה.

### סצנה 3 (~8ש) — צ'וזי + הדואל [צבע חוזר]
- רקע: קרם `#fbf5ea`. הצבעים חוזרים.
- **לוגו צ'וזי** נכנס במרכז עם אנימציית **Pop** (הסימן + המילה CHOOSY בכתום), ואז זז לפינה.
- הטלפון מציג **דואל**: שתי חולצות זו לצד זו (אייקוני חולצה). מצביע/יד נוגע באחת
  (חפש "tap"/"cursor"). זוג מתחלף — עשה **2-3 דפים קצרים** (round 1/2/3) עם מעבר מהיר.
- בסיבוב האחרון: חולצה מנצחת **ירוקה** עם **וי** (checkmark) וכפתור כתום קטן.
- אנימציות: Pop/Rise, מעברי Dissolve מהירים בין הסיבובים.

### סצנה 4 (~6ש) — משלוח + שמחה [צבע מלא]
- **קופסת משלוח** נפתחת (חפש "delivery box", "open box").
- **חולצה ירוקה בוהקת** קופצת החוצה (Pop).
- ילד/פרצוף שמח (חפש "happy kid" illustration) + **קונפטי מונפש** (חפש "confetti").
- צבע וחמימות מלאים.
